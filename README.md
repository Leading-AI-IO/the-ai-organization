## 📖 概要

BCGは言った。AIがもたらす価値の **70%は、人と組織のリデザインから生まれる** と。

しかし今、ほぼ全ての企業が残り30%——テクノロジーとデータ——に投資を集中させている。最新のLLMを導入し、データパイプラインを整備し、PoCを量産する。そして「AI導入率92%」という数字をボードに報告する。

だが、CFOの問いには答えられない。**「それで、いくら稼いだのか？」**

AI導入が失敗する原因は、技術にはない。組織にある。評価制度がAI時代の価値創造と噛み合わない。フィードバックの仕組みが存在しない。高価値人材に情報も権限も与えず、スコープだけを際限なく広げ、成果が出なければ個人のスタンスの問題に帰責する。

**本書は、特定の企業への批判ではない。構造の解剖だ。**

DeNA・Klarna・Shopify・NVIDIA・JPモルガン・Salesforce・サイバーエージェント・ソフトバンク——世界の先端事例と、日本企業固有の構造的病理を並べながら、「AI時代の組織をどう設計すべきか」という問いに正面から向き合う。
